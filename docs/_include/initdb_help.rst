Actions
^^^^^^^

``-v``, ``--vacuum``
    Vacuum an existing database.

``-b``, ``--backup`` *PATH*
    Vacuum into backup file (SQLite only).

``-r``, ``--rotate``
    Rotate completed tasks to new database (SQLite only).

``-t``, ``--truncate``
    Truncate database (task metadata will be lost).

Options
^^^^^^^

``-y``, ``--yes``
    Auto-confirm action (default will prompt).
